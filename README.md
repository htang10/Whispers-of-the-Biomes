## üåø Overview

Whispers of the Biomes is an immersive web experience that lets users explore five unique biomes - Forest, Mesa, Caldera, Marine, and Tundra.
Each biome features ambient sound, interactive elements, and short descriptive content designed to create a calm, engaging learning environment.

This project is built using HTML, CSS, and JavaScript (no external libraries/frameworks).

## ‚ú® Features

- A cinematic introductory experience that sets the atmosphere for the journey.
- A fully interactive 3D Earth that lets users rotate and explore the planet before selecting a biome.
- Smooth biome navigation with click, scroll, keyboard, and touch-swipe support.
- Five distinct biome environments: Forest, Mesa, Caldera, Marine, and Tundra.
- Ambient nature sounds that enhance immersion across all biomes.
- Unique splash screens for every biome that introduce each environment‚Äôs mood.
- High-resolution biome artwork presented in a horizontally scrollable, auto-centered layout.
- Interactive blur markers that reveal short educational descriptions when tapped or clicked.
- Smart, auto-positioning info popups that stay readable on all devices.
- A consistent interface across all biomes, including navigation controls and a quick return to the homepage.
